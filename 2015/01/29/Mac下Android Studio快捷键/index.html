Mac下Android Studio快捷键

  • Android Studio
  • Mac
  • Tools

从Eclipse转Android Studio有一阵子了，从最初的种种不习惯，到如今的爱不释手真的只有自己体验了才知道，如今我已经再也不想用Eclipse了有木有，哈哈！

Preferences -> Keymap 然后就可以选择你想要的快捷键，这里可以选择多种，如 Eclipse、Emacs、Net Beans 等编辑器的快捷键

  • 我自己选择的是 Mac OS X 10.5 +

以下是个人觉得非常实用的快捷键

Action Mac OSX
注释代码(//) Cmd + /
注释代码(/**/) Cmd + Option + /
格式化代码 Cmd + Option + L
清除无效包引用 Option + Control + O
查找 Cmd + F
查找+替换 Cmd + R
上下移动代码 Option + Shift + Up/Down
删除行 Cmd + Delete
扩大缩小选中范围 Option + Up/Down
快捷生成结构体 Cmd + Option + T
快捷覆写方法 Ctrl + O
快捷定位到行首/尾 Cmd + Left/Right
折叠展开代码块 Cmd + Plus,Minus
折叠展开全部代码块 Cmd + Shift + Plus,Minus
文件方法结构 Cmd + F12
查找调用的位置 Control + Option + H
大小写转换 Cmd + Shift + U
跳到调用位置 Cmd + B
查看当前方法声明 fn + F1
抽取方法 Cmd + Option + M
if、while、try Cmd + Option + T
补全方法/下一行 Cmd + shift + Enter
合并为一行 Option + Cmd + J
查看方法参数 Cmd + P
自动补全变量 Cmd + Option + V
查看当前类结构 fn + Control + H

(PS: 以上快捷键没有在win平台尝试确认，欢迎反馈错误与补充)
